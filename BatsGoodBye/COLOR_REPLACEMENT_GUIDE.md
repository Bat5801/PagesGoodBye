# 色彩系统调整说明文档

## 一、调整背景

为配合中考场景的情感铺垫，将原设计中的"黑红色系"调整为"白红色系"配色方案，提升视觉舒适度，营造积极向上、充满活力的视觉氛围。

## 二、色彩替换对照表

| 序号 | 元素类型 | 原黑红色系色值 | 新白红色系色值 | 应用位置 |
|:---:|---------|-------------|------------------|----------|
| 1 | 背景渐变起始色 | `#1a0a0a` | `#ffffff` | `body` 背景渐变 |
| 2 | 背景渐变结束色 | `#3a1a1a` | `#fff5f5` | `body` 背景渐变 |
| 3 | 主文字颜色 | `#e63946` | `#333333` | 正文文字 |

## 三、白红色系色值说明

### 背景色调
- **`#ffffff`** - 纯白（背景起始色）
- **`#fff5f5`** - 淡红（背景渐变结束色）
  - 用途：页面背景、容器背景
  - 特点：明亮舒适，提升视觉舒适度

### 主色调
- **`#e63946`** - 珊瑚红（主强调色）
  - 用途：标题文字、边框、图标、装饰元素
  - 特点：鲜艳而不过于刺眼，适合作为主视觉焦点

### 辅助色调
- **`#c1121f`** - 深红（次强调色）
  - 用途：时间数字、标签文字
  - 特点：明度较低，用于层级区分

### 文字色调
- **`#333333`** - 深灰（正文文字）
  - 用途：正文内容、段落文字
  - 特点：确保良好的可读性和对比度

## 六、开发实现指南

### CSS变量建议（可选优化方案）

若需进一步提升可维护性，建议引入CSS变量：

```css
:root {
    /* 白红色系变量 */
    --primary-red: #e63946;
    --secondary-red: #c1121f;
    --bg-light: #ffffff;
    --bg-light-pink: #fff5f5;
    --text-primary: #333333;
    --bg-content: rgba(255, 255, 255, 0.95);
    --bg-accent: rgba(230, 57, 70, 0.1);
    --shadow-weak: rgba(230, 57, 70, 0.3);
    --shadow-medium: rgba(230, 57, 70, 0.5);
}
```
